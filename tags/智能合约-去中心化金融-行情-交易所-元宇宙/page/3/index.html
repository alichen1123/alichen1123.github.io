什么是ERC20？

ERC20是Ethereum Request for Comments（ERC）系列标准中的一个，它定义了一套代币的接口和规范，使得在以太坊网络上创建和管理代币变得更加容易和标准化。简单来说，ERC20就像是一套指南，告诉开发者如何创建一种可以在以太坊上自由流通的数字资产。

2. 代币名称（string）

4. 总供应量（uint256）
