1. 什么是加密货币衍生品

2. 加密货币衍生品的类型

2.1 期货合约

期货合约是一种在未来特定日期以预定价格买卖加密货币的协议。买卖双方同意在未来某个时间点，按照合同约定的价格买入或卖出一定数量的加密货币。期货交易为投资者提供了对冲价格风险和锁定未来价格的工具。

2.2 期权合约

期权合约赋予购买者在未来某个时间点以特定价格买入或卖出加密货币的权利，但并非义务。买方可以行使这个权利，也可以选择不执行，而卖方则有义务在买方行使权利时履行合约。

2.3 差价合约（CFD）

2.4 永续合约

永续合约是一种没有到期日的期货合约，它通过资金费率来维持价格与标的资产价格的同步。永续合约为投资者提供了无限制的杠杆交易，但需要持续关注资金费率以避免清算风险。
