1. 工作量证明（Proof of Work, PoW）

优点

  • 去中心化：任何人都可以通过计算能力参与，无须信任。
  • 安全性高：需要大量计算力才能改变历史记录，抵抗51%攻击。

缺点

  • 能源消耗大：大量计算导致高能耗，对环境影响大。
  • 效率低：确认时间长，交易速度慢。
  • 集中化风险：算力集中可能导致中心化。

2. 权益证明（Proof of Stake, PoS）

优点

  • 节能：不需要大量计算，仅需持有代币即可。
  • 效率高：确认速度快，交易量大。
  • 抗审查性：攻击者需要持有大量代币，成本高。

缺点

  • 中心化风险：富有的节点可能控制网络。
  • 治理问题：如何公平分配权益可能引发争议。
  • 51%攻击：理论上仍有可能，但难度大。

优点

  • 高效：由代表节点进行验证，确认速度快。
  • 节能：减少计算，降低能耗。
  • 治理透明：选民可以投票选出代表。

缺点

  • 中心化：代表节点可能形成寡头。
  • 治理风险：选民可能被操纵。

EOS和Tron都是DPoS的典型应用。

缺点
