3. 避开高峰时段

三、优化交易大小

4. 分批交易

如果你打算进行大额交易，可以将其拆分成多个小交易。这样可以降低每次交易的矿工费，但要注意总费用可能会因为交易次数增多而增加。

四、选择合适的交易对

5. 选择低交易费的交易对

五、利用交易聚合器

6. 利用交易聚合器
