1. 零知识证明（Zero-Knowledge Proof, ZKP）

零知识证明是一种数学技术，允许一方（证明者）向另一方（验证者）证明某个陈述的真实性，而无需透露任何与该陈述相关的信息。在区块链中，ZKP可以用于验证交易的有效性，同时保护交易者的身份信息。例如，Zcash和zcash shielded transactions就是利用ZKP技术实现交易隐私的。

2. 环签名（Ring Signature）

环签名是一种多签名技术，使得签名者可以在一组签名者中匿名签名。在这种技术中，接收者只能确认签名来自一组签名者，但无法确定具体是哪一位。门罗币（Monero）是使用环签名技术的代表，它允许用户在交易中保持匿名。

3. 同态加密（Homomorphic Encryption）

同态加密是一种加密技术，允许在数据加密状态下进行计算，而无需先解密。在区块链中，同态加密可以用于保护交易数据的隐私，使得数据在被处理时仍然保持加密状态，而不会泄露敏感信息。比如，以太坊的zk-EVM项目正在探索使用同态加密来保护智能合约的隐私。

4. 混合服务（Mixer）

混合服务，也称为混币服务，通过将用户的资金与其他用户的资金混合，以混淆资金的来源和去向，实现匿名交易。Tornado Cash是比特币和以太坊上最知名的混合服务之一，它通过复杂的算法，使得资金流动的可追踪性大大降低。
