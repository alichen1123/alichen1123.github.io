2.2. Liquid网络

Liquid网络则利用了中继网络和通道技术，允许参与的交易所之间直接转移比特币。每个交易所都维护一个在Liquid侧链上的“主账户”，通过中继节点，交易所可以在不通过主链的情况下转移比特币。为了保证安全，Liquid网络使用了多签名和时间锁定功能，确保资金只能在指定的时间内被转移。

DAG，直译为有向无环图，是一种非线性的数据结构。在传统的区块链中，交易被组织成一个个区块，形成一条线性的链。而在DAG中，交易不再是线性排列，而是以网状结构相互连接，每个交易可以同时依赖多个前交易，形成一个复杂的网络。这种结构极大地提升了数据处理的并行性，降低了交易确认时间。
