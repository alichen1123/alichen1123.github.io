以太坊的EIP-1559：燃烧交易费用，稳定网络

EIP-1559简介

EIP-1559（Ethereum Improvement Proposal 1559）是2021年以太坊网络升级的重要组成部分，旨在通过引入基fee和燃烧机制，改变网络的交易费用动态，从而减少费用波动，提高用户体验。

基fee与燃烧机制

在EIP-1559中，每次交易时，用户需要支付一个固定的基fee，这个费用会根据网络拥堵程度动态调整。如果网络拥堵，基fee会上涨，交易费用会更高；反之，网络空闲时，基fee会下降，交易费用也会随之降低。更重要的是，这部分基fee会被直接烧掉，从流通中永久移除，减少了以太坊的总供应量。

EIP-1559的影响

EIP-1559的实施显著降低了交易费用的波动性，减少了“矿工费竞价”现象，使得用户能更好地预测交易成本。然而，这也可能导致以太坊的通胀率下降，对矿工收入产生影响，促使他们寻找其他收入来源。

Algorand的纯量证明：去中心化与费用透明

纯量证明（PurePoS）简介

Algorand是一种基于纯量证明的共识机制的区块链，它通过一种创新的算法，使得网络在保持高吞吐量的同时，保持了较低的交易费用和能源消耗。在Algorand中，交易费用是固定的，且由发送方预先支付，确保了费用的透明性和可预测性。

纯PoS与费用模型

纯PoS机制使得Algorand的交易费用不依赖于网络拥堵程度，而是由网络算法根据当前的交易量自动设定。这种机制避免了像EIP-1559那样的燃烧机制，从而保持了代币的总供应量稳定。

纯PoS的优势

Algorand的纯PoS费用模型提供了稳定的交易费用和快速的交易确认时间，这使得它在处理大规模交易时更具优势。此外，由于交易费用固定，用户可以更好地规划和控制成本，提高了网络的可预测性。

什么是自动做市商（AMM）

在传统的中心化交易所中，做市商是买卖双方的中间人，他们提供流动性，确保市场始终有买卖订单。而在去中心化交易所（DEX），AMM扮演了做市商的角色，但没有中心化机构，而是通过算法自动进行。

AMM的核心是流动性池，它包含两种或多种加密货币，如Uniswap中的ETH和USDC。用户可以将资产存入流动性池，换取平台代币作为奖励。当其他人交易时，AMM会根据算法自动调整价格，以维持池中资产的相对比例，无需人工干预。

AMM的运作原理：恒定乘积公式

AMM最著名的算法是恒定乘积公式，以Uniswap为例，其公式是：

[ x \cdot y = k ]

其中，( x ) 是一种资产的量，( y ) 是另一种资产的量，而 ( k ) 是一个常数。当用户买卖其中一种资产时，另一种资产的量会自动调整，以保持乘积 ( k ) 不变。这种机制保证了价格的连续性，但可能会产生与中心化交易所不同的价格。

  1. 价格发现：价格可能与中心化交易所存在差异，特别是在低流动性情况下。

EIP-1559是 Ethereum Improvement Proposal（以太坊改进提案）中的一项，旨在解决以太坊网络上的交易费用波动问题。该提案于2021年8月5日实施，引入了新的费用结构，包括基础费用（Base Fee）、优先费（Tip）和燃烧机制。

基础费用与优先费

基础费用是用户为确保交易被矿工打包的最低费用，它会根据网络拥堵程度自动调整。优先费则是用户为了优先处理交易而额外支付的费用。在EIP-1559中，基础费用会在每个区块中被烧毁，减少了以太坊的总供应量，理论上有助于以太坊价格的稳定。

在BSC上，交易费用的计算是基于竞拍机制，用户需要设置Gas Price来竞争矿工的处理优先级。矿工通常会选择Gas Price较高的交易进行打包，这导致了交易费用的波动性。

费用退款

与EIP-1559不同，BSC的Gas费用不会被烧毁。如果交易未被确认，用户可以申请退款，这在一定程度上降低了用户的交易成本风险。

EIP-1559（ Ethereum Improvement Proposal 1559）是2021年以太坊网络升级的核心部分，旨在解决交易费用的波动性和不确定性。这个新机制引入了基础费用（Base Fee）和可变费用（Tip），并将基础费用的一部分直接烧掉，从而减少流通中的以太币供应，以应对网络拥堵时的费用飙升。

15.2 EIP-1559的运作机制

  • 基础费用：用户在发送交易时需要支付的最低费用，由市场动态决定。
  • 可变费用（小费）：用户可以选择支付额外费用以加快交易确认速度。
  • 燃烧机制：每当网络拥堵时，基础费用的一部分会被永久烧毁，减少流通中的ETH供应，以平衡供需关系。

15.3 EIP-1559的影响

  • 费用透明性：基础费用的透明度提高了用户对交易成本的预期。
  • 供需调节：燃烧机制有助于稳定网络费用，减少“矿工费竞赛”。
  • 通缩预期：长期来看，燃烧机制可能导致以太币供应减少，可能提升其价值。

二、Cardano的雪崩（Ouroboros）费用模型

2.1 Cardano简介

Cardano是一个以科学和学术研究为基础的区块链平台，其费用模型是其核心创新之一。Cardano采用的是权益证明（Proof-of-Stake, PoS）共识机制，与以太坊的燃烧机制不同，Cardano的费用模型更侧重于网络的稳健性和长期可持续性。

2.2 雪崩（Ouroboros）费用模型

在Cardano网络中，交易费用由验证者（Stake Pool）收取，而不是矿工。费用由网络自动调整，以确保交易的即时确认和网络的稳定运行。Cardano的费用模型旨在提供稳定、可预测的交易成本，同时鼓励用户参与网络的共识过程。

2.3 Cardano费用模型的影响

  • 稳定费用：Cardano的费用模型旨在保持费用的稳定，避免以太坊中可能出现的极端波动。
  • 参与激励：用户通过参与验证过程，既能赚取交易费用，又能为网络安全做出贡献。
  • 可持续性：Cardano的费用模型支持长期的网络发展，减少对环境的影响。

三、EIP-1559与Cardano费用模型的比较

3.1 费用透明度

EIP-1559通过基础费用提供透明度，而Cardano的费用由验证者设定，但网络会自动调整以保持稳定。

3.2 网络效率

EIP-1559的燃烧机制有助于快速处理交易，而Cardano通过验证者网络确保即时确认。

3.3 可持续性

Cardano的PoS机制和费用模型更注重长期可持续性和环保，而EIP-1559的燃烧机制可能导致通缩预期。

四、结论：革新与稳健的权衡
