3. 混币（Coin Mixing）

4. 同态加密（Homomorphic Encryption）

5. 混合网（Mixing Networks）

混合网，也称为混淆网，是一种网络架构，通过多层中继节点，将用户数据进行混淆，提高匿名性。Tor网络和I2P（Invisible Internet Project）就是混合网的典型应用。
