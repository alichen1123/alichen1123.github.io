2. 零知识证明（Zero-Knowledge Proofs, ZKP）

零知识证明是一种神奇的技术，它允许一方（证明者）向另一方（验证者）证明自己知道某个信息，而无需透露具体信息。在区块链中，ZKP可以用来验证交易的有效性，而无需展示交易的详细内容。比如，Zcash就是利用ZKP技术，实现了交易金额的隐藏，只显示交易的发生。

3. 混合服务（Mixer）

混合服务，也称为 tumbling 或者 laundry，是一种将多个用户的加密货币混合在一起的服务，以混淆资金的来源和去向。用户通过混合服务，将资金与其他人资金混合，从而匿名化交易。Tornado Cash就是这样的一个知名的混合服务。

4. 多重签名（Multi-Signature, Multisig）

多重签名技术要求一笔交易需要多个私钥的签名才能完成，增加了资金控制的复杂性，也提高了隐私保护。例如，比特币的Segwit（Segregated Witness）技术就支持多重签名，提高了交易的隐私性。
