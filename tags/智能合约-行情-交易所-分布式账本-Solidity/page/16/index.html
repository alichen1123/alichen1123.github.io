什么是Layer 2？

Layer 2，顾名思义，是构建在以太坊主链（Layer 1）之上的第二层网络，它们在不影响主链安全性的前提下，通过优化交易处理方式，实现更高的交易吞吐量和更低的交易成本。Layer 2方案通常分为两种类型：状态通道和侧链。

1. 状态通道

状态通道是一种点对点的解决方案，它通过在两个参与者之间建立一个专用的通道，将交易从主链转移到通道中进行，从而避免频繁的主链交易。最知名的代表是闪电网络（Lightning Network），它在比特币网络上实现了类似的功能。以太坊的闪电网络实现包括Optimism 和 Loopring 等。

Optimism

Optimism通过一个名为“optimistic rollup”的技术，允许用户在链下进行大量交易，然后以批量的形式提交到主链上验证。如果在设定的时间内没有争议，交易就被确认，大大提高了交易速度。

Loopring

Loopring则采用了一种名为“零知识证明”的技术，使得交易在链下进行，同时保持了数据的隐私性。通过这种方式，Loopring能够实现低延迟和低成本的交易。

2. 侧链

侧链是一种独立的区块链，与主链通过特定的协议进行通信。侧链可以独立处理交易，然后将结果提交到主链进行确认。侧链的代表有Polygon（原Matic Network） 和 Arbitrum。

Polygon（原Matic Network）通过提供一个独立的以太坊扩展网络，实现了快速且低成本的交易。Polygon使用了** zk-Rollup** 技术，通过零知识证明来验证链下交易的正确性。

Arbitrum
