什么是Layer 2和侧链？

侧链

  2. Relay机制：Relay机制通过在不同链间建立信使节点，这些节点负责将消息从一条链传递到另一条链。Relay机制简单易懂，但可能面临数据延迟和节点故障的问题。

  3. 侧链与中继链（Sidechains and Bridging）：侧链是与主链平行运行的链，它们之间通过中继链进行连接。侧链可以拥有自己的共识机制，而中继链负责验证和传递跨链交易。

  4. 跨链通信协议（Interledger Protocol, ILP）：ILP是一种通用的跨链通信协议，它允许任意两个独立的账本系统进行通信，通过“支付通道”实现资产的跨链转移。
