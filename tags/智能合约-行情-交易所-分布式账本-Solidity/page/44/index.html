5. 地址混淆（Address Mixing）

原子交换是最早的跨链交易方式，它通过一组不可分割的交易来确保资产在两个链上的转移是同时发生的，如果其中一方失败，所有交易都将被撤销，保证了交易的原子性。这种技术适用于同构链（基于相同共识机制的链）之间的交易。

2. 中继网络

中继网络通过在不同链之间建立桥梁节点，这些节点负责资产的转移和验证。比如，Polkadot和Cosmos等项目就采用了中继网络来实现跨链通信。这种方式支持异构链（基于不同共识机制的链）之间的交互，但依赖于网络中的特定节点，可能带来一定的中心化风险。

3. 侧链

侧链技术允许在主链之外创建独立的区块链，这些侧链可以与主链进行资产的互换。侧链的典型代表是比特币的闪电网络和以太坊的雷电网络。侧链可以提供更快的交易速度和更低的费用，但同样需要解决安全性和互操作性问题。
