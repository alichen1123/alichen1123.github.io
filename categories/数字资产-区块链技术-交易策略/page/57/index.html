1. 按照所有权类型划分

1.1 热钱包（Hot Wallet）

热钱包是连接到互联网的，方便快捷，但安全性相对较低。它们通常作为在线钱包或移动应用存在，如Coinbase、Binance等交易所提供的钱包。热钱包适合日常交易和小额资产的管理。

冷钱包则不直接连接互联网，安全性更高。常见的冷钱包有硬件钱包（如Trezor、Ledger）和纸钱包。冷钱包适合存储大量价值的加密货币，或长期持有。

托管钱包由第三方服务提供商管理你的私钥，如交易所钱包。优点是操作简便，缺点是你的资产受第三方控制，存在潜在风险。

2.2 非托管钱包（non-custodial wallet）

非托管钱包，也称自主管理钱包，你完全掌控私钥。硬件钱包和纸钱包都属于此类。非托管钱包确保了资产的完全所有权，但同时也要求用户对私钥安全负责。

通用钱包支持多种加密货币的存储，如MyEtherWallet、Exodus。它们适合持有多种资产的投资者。

3.2 专用钱包（Single-Coin Wallet）

专用钱包只支持特定的加密货币，如Bitcoin Core（比特币）或MetaMask（以太坊）。这类钱包专注于特定币种，功能可能更专业。

4. 按照用户友好度划分

4.1 用户友好型钱包

以太坊的EIP-1559：燃烧交易费用，稳定网络

EIP-1559简介

EIP-1559（Ethereum Improvement Proposal 1559）是2021年以太坊网络升级的重要组成部分，旨在通过引入基fee和燃烧机制，改变网络的交易费用动态，从而减少费用波动，提高用户体验。

基fee与燃烧机制

在EIP-1559中，每次交易时，用户需要支付一个固定的基fee，这个费用会根据网络拥堵程度动态调整。如果网络拥堵，基fee会上涨，交易费用会更高；反之，网络空闲时，基fee会下降，交易费用也会随之降低。更重要的是，这部分基fee会被直接烧掉，从流通中永久移除，减少了以太坊的总供应量。

EIP-1559的影响

EIP-1559的实施显著降低了交易费用的波动性，减少了“矿工费竞价”现象，使得用户能更好地预测交易成本。然而，这也可能导致以太坊的通胀率下降，对矿工收入产生影响，促使他们寻找其他收入来源。

Algorand的纯量证明：去中心化与费用透明

纯量证明（PurePoS）简介

Algorand是一种基于纯量证明的共识机制的区块链，它通过一种创新的算法，使得网络在保持高吞吐量的同时，保持了较低的交易费用和能源消耗。在Algorand中，交易费用是固定的，且由发送方预先支付，确保了费用的透明性和可预测性。

纯PoS与费用模型
