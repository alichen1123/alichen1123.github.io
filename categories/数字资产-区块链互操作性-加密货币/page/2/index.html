跨链资产托管协议的工作原理

哈希锁定是最早被提出的跨链协议之一。在源链上，用户将资产锁定，并生成一个包含资产信息的哈希值。在目标链上，接收方需要提供一个能够解开源链上哈希的密钥，才能获取资产。这个过程确保了资产只能在双方都同意的情况下转移。

2. 时间锁定（Time Locking）

3. 中继网络（Relay Network）

中继网络是一种分布式网络，它在不同链之间传递信息和交易。中继节点负责验证交易的有效性，并在各链之间传递哈希锁定或时间锁定的信息，确保资产安全转移。

4. 原子交换（Atomic Swaps）

原子交换是一种无需信任第三方的点对点交易方式。在交易过程中，如果一方未完成交易，另一方的资产将自动退回，保证了交易的原子性。

3. 侧链与中继

侧链是与主链平行运行的区块链，它们之间通过中继机制实现资产的转移。侧链可以独立运行，而中继则负责在主链和侧链之间传递信息。Polkadot和Cosmos是侧链与中继技术的代表。

4. 跨链通信协议

这类技术通过制定统一的通信协议，让不同链上的智能合约能够相互通信。如Wormhole协议允许以太坊和Solana之间的资产互换。
