一、加密货币概述

1.1 什么是加密货币

加密货币，简单来说，是一种基于区块链技术的数字货币，以比特币（Bitcoin）为代表。它们使用复杂的加密算法确保交易的安全性和匿名性，且不受任何中央机构的控制。

1.2 区块链技术

区块链是加密货币的核心技术，它是一个去中心化的分布式账本，记录了所有交易历史。每个区块包含一组交易，通过哈希函数链接形成链式结构，确保数据不可篡改。
