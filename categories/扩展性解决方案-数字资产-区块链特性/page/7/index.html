什么是ERC20？

ERC20，全称为“Ethereum Request for Comments 20”，是基于以太坊（Ethereum）区块链上的一种代币标准。它定义了一套代币发行、转移、持有和操作的规则，使得开发者可以轻松地在以太坊网络上创建和管理自己的代币。ERC20标准的出现，使得代币发行变得简单、快速，也极大地推动了以太坊生态的繁荣。

ERC20的四大组件

ERC20标准主要由四个关键组件构成：

  1. 合约地址（Address）：每个ERC20代币都有一个唯一的合约地址，它在以太坊区块链上标识了代币的智能合约。

  4. 总供应量（Total Supply）：表示代币的总发行量，这个数值在创建后通常不可更改。

ERC20的主要函数

  1. totalSupply()：返回代币的总供应量。

  4. transferFrom(address _from, address _to, uint256 _value)：从一个地址向另一个地址转移代币，需要原地址的批准。
  5. allowance(address _owner, address _spender)：查询指定地址的代币转移批准额度。
  6. approve(address _spender, uint256 _value)：批准指定地址转移指定数量的代币。
  7. transferEvent(address indexed _from, address indexed _to, uint256 _value)：事件触发，记录代币转移。

以太坊，作为智能合约的摇篮，其Layer 2扩展方案是其应对高交易量和拥堵问题的重要手段。Layer 2方案并不改变以太坊主链的共识机制，而是通过在主链之外构建额外的层来处理交易，从而减轻主链的负担。最知名的Layer 2方案包括Optimism、Arbitrum和Polygon等。

Optimistic Rollup

Optimistic Rollup是Layer 2中的一种，它通过在链下批量处理交易，然后将结果打包成一个较小的证明上传到主链，从而实现高吞吐量。如果在一定时间内没有人挑战这个证明，那么交易就被确认有效。这种机制牺牲了一定的即时性，但显著提高了交易速度和降低了费用。

ZK-Rollup
