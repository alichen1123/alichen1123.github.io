1. 零知识证明（Zero-Knowledge Proofs, ZKP）

零知识证明是一种密码学技术，它允许一方（证明者）向另一方（验证者）证明某件事情的真实性，而无需透露任何有关该事情的信息。在区块链中，零知识证明被用来验证交易的有效性，同时隐藏交易的详细内容。典型的例子有Zcash和Monero，它们利用 zk-SNARKs（零知识简洁非交互式论证）技术，实现了交易的匿名性。

2. 环签名（Ring Signatures）

环签名是一种多签名技术，它使得消息的发送者可以隐藏在一组签名者（环）中，接收者只能确认签名的有效性，而无法确定具体是哪个签名者。Dash和Komodo等项目利用环签名技术，使交易者在不暴露身份的情况下进行交易。

3. 混币（Coin Mixing）

混币技术，也称为“币混淆”，通过将多个用户的资金混合在一起，使得追踪单个资金来源变得困难。TumbleBit和Wasabi Wallet等项目通过创建临时的、不可追踪的代币，实现了交易的匿名性。

4. 同态加密（Homomorphic Encryption）

同态加密是一种特殊的加密技术，允许在加密数据上直接进行计算，而无需先解密。在区块链中，同态加密可以保护交易数据的隐私，同时允许在链上进行计算。例如，zkSync和Plonk等项目利用同态加密技术，实现了隐私保护的智能合约。

5. 混合服务（Mixing Services）

混合服务，也被称为“混币服务”，是一种第三方服务，用户可以将资金发送到混合服务，然后由服务将资金重新分配给多个地址，以混淆资金的来源和去向。典型的混合服务包括Wasabi、CoinJoin和Tornado Cash。
