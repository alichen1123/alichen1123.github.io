优点

2. 权益证明（Proof of Stake, PoS）

优点

  • 能源效率：无需大量计算，仅需锁定资产，显著降低能耗。
  • 更快确认：共识过程更高效，交易确认时间短。

缺点

  • 能源效率：与PoS类似，无需大量计算。

缺点

  • 潜在的中心化：代表节点可能形成小集团，影响网络决策。
  • 信任问题：选民需信任代表节点，可能存在滥用权力的风险。

4. 实用拜占庭容错（Practical Byzantine Fault Tolerance, PBFT）

优点

  • 高吞吐量：PBFT在小规模网络中具有极高的交易处理能力。
  • 快速确认：共识过程快速，交易确认时间短。

缺点

  • 扩展性：随着节点数量增加，性能下降，不适合大规模网络。
  • 信任模型：需要预选节点，可能增加中心化风险。
