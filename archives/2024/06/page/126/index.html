什么是ERC20？

ERC20是Ethereum Request for Comments（以太坊请求提案）的20号提案，它是以太坊上发行和管理代币的一种标准。ERC20标准定义了一套通用接口，使得开发者能够轻松创建、发行和管理自己的代币，这些代币可以在以太坊网络上自由流通。这种标准化的做法简化了代币的创建过程，同时也确保了代币在各种应用和钱包之间的互操作性。

ERC20的主要功能

  2. 余额查询：通过balanceOf函数，可以查询某个地址的代币余额。
  3. 允许代币转移：通过allowance和approve函数，持有者可以授权他人代表自己转移代币。
  4. 事件通知：代币的转移和批准等操作会触发事件，如Transfer和Approval，以便其他智能合约或应用程序接收并处理这些事件。

什么是Rollups？

zk-rollups与optimistic rollups：基本原理

zk-rollups（零知识Rollups）

zk-rollups利用零知识证明（Zero-Knowledge Proofs，ZKPs）技术来验证交易的正确性。ZKPs允许验证者在不揭示具体信息的情况下证明某个陈述的真实性。在zk-rollups中，第三方（称为rollup运营商）将交易打包成一个rollup，然后生成一个ZKP来证明这些交易的正确性。只有当有人质疑这个证明时，才会在主链上执行完整的验证过程。由于ZKPs的特性，这种验证过程是高度隐私保护的。

optimistic rollups（乐观Rollups）

optimistic rollups基于一个假设：网络中的大多数参与者是诚实的。在optimistic rollups中，交易被打包成rollup，然后直接在主链上发布，假设所有交易都是有效的。如果有人发现交易中有欺诈行为，他们可以在一定时间内提交一个欺诈证明，触发所谓的“挑战期”。在挑战期内，主链上的所有节点会重新验证交易，如果欺诈被证实，交易会被撤销，恶意参与者将面临惩罚。
